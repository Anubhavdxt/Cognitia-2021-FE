# Cognitia-2021-FE

## Front-end of the website for **Cognitia** (technical festival of NIT Meghalaya)

## Run Locally

Clone the project

```bash
  git clone https://github.com/Coding-Club-NIT-Meghalaya/Cognitia-2021-FE
```

Go to the project directory

```bash
  cd Cognitia-2021-FE
```

Install live-server

```bash
  npm install live-server
```

Start the server

```bash
  live-server
```
